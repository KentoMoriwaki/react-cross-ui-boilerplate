# react-cross-ui-boilerplate


## Build

```
yarn run build # build-ios | build-android
# Or watch
yarn run start # start-ios | start-android
```

## Examples

Before build examples, setup symlinks

```
yarn link
```

### Web

```
cd examples/web
yarn install
yarn link react-cross-ui-boilerplate
yarn start
```

### Native

```
cd examples/native
yarn install
yarn link react-cross-ui-boilerplate
yarn start
```
